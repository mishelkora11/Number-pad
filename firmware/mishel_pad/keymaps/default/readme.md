\# Default Macropad Layout



This is the default layout that comes flashed on my Macropad. For the most

part it's a straightforward and easy to follow layout. It has numbers 1 through 0 just like a usual numberpad. Although number 8 serves also as an up arrow if held down for 3 seconds. In the same way number 4 is a left arrow, number 6 is a right arrow and number 2 is a down arrow.
