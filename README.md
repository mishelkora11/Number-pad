Firmware:

The micropad uses QMK Firmware.


Description:

A custom handwired 10-key macropad running QMK firmware on a Seeed Studio XIAO RP2040 development board. 

* Keyboard Maintainer: [mishelkora11](https://github.com/mishelkora11)
* Hardware Supported: Seeed Studio XIAO RP2040, MX-style switches, and an 0.91 inch OLED Display.


BOM:

1x seeed studio rp2040

4x  style switches

1x 0.91 inch oled display

4x M3x5x4 Heatset inserts

1x Case 

4x M3x16mm screws
